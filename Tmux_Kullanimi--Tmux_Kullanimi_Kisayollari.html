Tmux Kullanimi Kisayollari

1. apt install tmux
2. tmux new -s TEST
3. ctrl + r → research in history
4. ctrl + b = Prefix key
5.Prefix key + c = create new window
6. Prefix Key + 0 → sıfırıncı pencere, Prefix Ket +1 birinci pencere, yani pencereler arası geçiş prefix key + numaralar şeklinde yapılmaktadır.
7. exit ile windowlar kapatılmaktadır.
8. Prefix Key + , → rename window
9. Prefix Key + alt shift + p -→ log
10. Prefix Key + " (yatay ayırma)
11. Prefix Key + % (dikey ayırma)
12. Prefix Key + ok tuşları page'ler arası geçiş
13. Prefix Key +z (zoom in ), aynı şekilde Prefix Key +z (zoom out)
14. Prefix Key y'e basılı tutatrken okları kullanır isek page leri genişletip daraltabiliriz
15. Prefix Key + { pane saat yönüde döndürür
16. Prefix Key + } pane i saat yönünün tersine döndürür
17. Prefix Key + space pane şeklini değiştiri
18. alt + . (kelime kelime historiyde geriye gider)
19. ctrl + a (begginging of the line)
20. ctrl + e (end of line)
21. ctrl + ok (kelime kelime geri ileri yapar)
22. Prefix Key + ? → bize neler yapabileceğimizi gösterir
23. Prefix Key + t → bize time i gösterir
24. prefix key + [ then / aşağıya doğru arama
25. prefix key + [ then ? yuarıya doğru arama

tmux ls (bağlandığımız sunucu üzerindeki tmux session'larını gösterir)
tmux attach -t NAME (attach tmux sessions)
prefix key +d (detach sessions)

https://github.com/tmux-plugins/tmux-logging // tmux log plugin'i

tmux'historisini kaydetmek için prefix key + alt + shift + p

/home/anaytemiz/.tmux.conf dosyası aşağıdaki gibidir.

anaytemiz@anaytemiz-ThinkPad-T470:~$ cat .tmux.conf 
#Remap Prefix to screens
set -g prefix C-a
bind C-a send-prefix

# Quality of life stuff
set -g history-limit 10000
set -g allow-rename off
set -g mode-keys vi
## Join Window

bind-key j command-prompt -p "join pane from:" "join-pane -s '%%'"
bind-key s command-prompt -p "send pane to:"  "join-pane -t '%%'"

# Search Mode VI (default is emac)
set-window-option -g mode-keys vi


set -g @plugin 'tmux-plugins/tmux-logging'

run-shell ~/clone/path/logging.tmux


artık prefix key'im ctrl + a oldu, editor ayarları vi'ye göre ayarlandı, log opsiyonu enable edildi.
